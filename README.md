# Agent Management (.agents)

A template for tracking development artifacts and AI-generated outputs.

## Folder Structure

- **`plans/`**: Implementation designs and technical specifications.
- **`walkthroughs/`**: Verification reports and proof-of-work documentation.
- **`shortcuts/`**: Prompting guidance and naming conventions for AI models.

## Workflow

When generating files, adhere to the formats defined in `shortcuts/`:
- **Plans**: `plan_[agent-name]_[topic]_[date]_[time].md`
- **Walkthroughs**: `walkthrough_[agent-name]_[topic]_[date]_[time].md`
