Development Services
- Start and manage the core development services (Frontend & Backend).
- Commands: 
  - Backend: `py manage.py runserver` (CWD: `backend/`)
  - Frontend: `npm run dev` (CWD: `frontend/`)
- Instructions:
  - When the user asks to "start dev", "run services", or "initialize environment", execute both commands in the background.
  - Monitor logs for startup success and report back to the user.
  - Reference shared guardrails: `.agents/shortcuts/SHARED_GUARDRAILS.md`
